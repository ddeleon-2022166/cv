# Curriculum Vitae

---

## Descripción

Aplicación web desarrollada con Spring Boot y Thymeleaf que permite la visualización de un CV, propio a Diego Alejandro de León García.
El sistema implementa Thymeleaf para el frontend del proyecto.

---

## Tecnologías Utilizadas

- Java 17+
- Spring Boot
- Thymeleaf
- Maven

---

## Requisitos Previos

Antes de ejecutar la aplicación, asegúrate de tener instalado:

- Java JDK 17 o superior
- Maven (opcional si usas Maven Wrapper)
- IDE recomendado (IntelliJ IDEA, Eclipse, VS Code)

---

## Ejecución

1. Clonar el repositorio:

   git clone <URL_DEL_REPOSITORIO>  
   cd gestordetareas

2. Ejecutar la aplicación:

   En Linux/Mac:
   ./mvnw spring-boot:run

   En Windows:
   mvnw.cmd spring-boot:run

3. Acceder a la aplicación:

   http://localhost:8081
